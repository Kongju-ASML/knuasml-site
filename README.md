# KNU ASML Website Preview

Static preview site for the Advanced Structural Materials Laboratory at Kongju National University.

## Local preview

```powershell
npm install
npm run dev
```

## Production build

```powershell
npm run build
```

The deployed preview URL is:

```text
https://knu-asml-preview.gasongknu.chatgpt.site/
```

## Current structure

- Home
- About ASML
- Research Areas
- How We Work
- Join ASML
- People
- Projects
- Facilities
- Data Infrastructure
- Publications
- News
- Contact

## Notes

The current version avoids unverified publication titles, member names, equipment ownership claims, and project-specific numbers. Those should be filled from confirmed lab records.
